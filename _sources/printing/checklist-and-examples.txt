Checklist and examples
======================


Checklist
---------
    * Remove doubles (:kbd:`W`, :menuselection:`Remove Doubles`)
    * Run 3D Print Toolbox checks (:menuselection:`Check all`)
    * Click buttons in output, evaluate problems
    * Check `Mesh Analysis <https://www.blender.org/manual/modeling/meshes/mesh_analysis.html>`_ as well
    * Try some `Mesh cleanup tools <https://www.blender.org/manual/modeling/meshes/cleanup.html>`_
    * Triangulate (ahead of time, to check for issues) (:kbd:`Ctrl-T`)
    * Re-run 3D Print Toolbox
    * Select object
    * :menuselection:`File --> Export --> Stl`

Examples
--------
* too much overhang
* wrong normals
* cancelled an extrusion, so need to remove doubles
* non-flat faces
