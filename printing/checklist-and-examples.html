Checklist and examples¶

Checklist¶

  • Remove doubles (W, Remove Doubles)
  • Run 3D Print Toolbox checks (Check all)
  • Click buttons in output, evaluate problems
  • Check Mesh Analysis as well
  • Try some Mesh cleanup tools
  • Triangulate (ahead of time, to check for issues) (Ctrl-T)
  • Re-run 3D Print Toolbox
  • Select object
  • File ‣ Export ‣ Stl

Examples¶

  • too much overhang
  • wrong normals
  • cancelled an extrusion, so need to remove doubles
  • non-flat faces
